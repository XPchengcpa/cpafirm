Hide specific menus for specific user

This module allows setting which menus specific users cannot see.

    Configure which menus to hide on the user page.

    You can see the same effect on the menu page.

    After the user (demo) logs into the system, they cannot see the hidden menu.

    Even if the user knows the URL of the menu, they do not have access to it.
