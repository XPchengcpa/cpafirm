Remove Powered by Odoo from Website & Add Custom Text - Complete Branding Control

Description

This free module gives you complete control over the "Powered by Odoo" footer branding. Hide the default message entirely or replace it with your own custom HTML content including text, links, colors, and formatting.

Perfect for branding, white-label solutions, and custom website footers. The module includes a rich HTML editor, toggle controls, and seamless integration with Odoo's website footer.

Features
Hide Branding

Completely hide "Powered by Odoo" footer

Rich HTML Editor

Custom footer with HTML formatting

Toggle Control

Easy show/hide functionality

Website Integration

Seamless footer integration

Configuration
Setting up Footer Customization
Go to the "Settings" - "General Settings" menu and find the "Powered by Odoo Settings" section. Configure your custom footer text and toggle the visibility of the default branding.
Apply the changes. The custom footer settings will be activated and visible on your website.
HTML Editor Features
The module includes a powerful HTML editor (CKEditor) that allows you to create rich, formatted footer content with:
  • Text Formatting: Bold, italic, underline, colors
  • Links: Internal and external URLs
  • Alignment: Left, center, right alignment
  • Lists: Bulleted and numbered lists
  • Special Characters: Symbols and emojis
Toggle Functionality
Use the toggle switch to show or hide the "Powered by Odoo" branding. When enabled (checked), the default branding is shown. When disabled (unchecked), the branding is hidden or replaced with your custom content.
This provides complete control over the visibility of Odoo branding without affecting other website functionality.
